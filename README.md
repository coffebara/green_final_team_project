## 설치 가이드

**요구 사항**
- Npm  
- Eclips EE (STS 4)  
- VsCode  
- Oracle 11g  

**초기화**  

    $ git clone https://github.com/coffebara/green_final_team_project.git   
    $ cd green_final_team_project

**Backend**  

    $ cd itdalearn-be   

**Fromtend**  

    $ cd itdalearn-fe   
    $ npm install   
    $ npm start   
 
## 주요 기능
